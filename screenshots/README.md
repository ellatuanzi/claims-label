# Screenshots

Please add the following screenshots to this folder:

1. **app2-input.png** - Screenshot of the single claim scoring interface with text input
2. **app2-results.png** - Screenshot showing the clarity/complexity scores and contributing factors
3. **app1-dashboard.png** - Screenshot of the batch analysis dashboard with charts

## How to Take Screenshots

### For App 2 (Single Claim Scoring):
1. Run `streamlit run app2.py`
2. Enter a sample claim note
3. Click "Score Claim"
4. Take screenshots of:
   - The input interface (before clicking button)
   - The results display (after scoring)

### For App 1 (Batch Analysis):
1. Run `streamlit run app.py`
2. Click "Run LLM Scoring on Dataset"
3. Wait for completion
4. Take a screenshot of the full dashboard with charts and statistics

## Recommended Screenshot Settings

- Resolution: 1920x1080 or higher
- Format: PNG
- Include browser chrome for context (optional)
- Capture full width of the Streamlit app
